#  CRUD Basic - Proyecto Spring Boot

##  Semana 1: Modelo y MER

###  Descripción
Proyecto CRUD básico con Spring Boot para gestión de cursos online.

###  Tecnologías
-  Java 17
-  Spring Boot 3.5.7
-  JPA/Hibernate
-  SQL Server

###  Estructura actual (Semana 1)
```
src/main/java/com/sena/crudbasic/
├── model/
│   ├── Curso.java       
│   ├── Instructor.java  
│   └── Leccion.java     
└── CrudbasicApplication.java
```

###  Completado en Semana 1
-  Modelos creados (Curso, Instructor, Leccion)
-  Diagrama MER documentado
-  Anotaciones JPA configuradas
-  Relaciones entre entidades definidas
